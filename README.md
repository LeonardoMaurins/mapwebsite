# Campus Information Application
A web application which allows for ease of navigation to first year students on campus amongst other useful tools.

### Prerequisites

* [PHP](https://www.php.net/) - Scripting Language
* [Composer](https://www.npmjs.com/) - Package Manager

### Running locally

1. Navigate to the project directory within a terminal.
2. Execute ```composer install``` for the project vendor.
3. Execute ```php -S localhost:8000 -t ./public``` to host the website.
4. Navigate to the localhost on your browser at ```localhost:8000```.

## Features

* Registration
* Login
* User & Admin Roles
* Campus Map Interface
* Bus Schedules
* Navigational staff directory with filters
* CRUD Implementation
* Codeception Tests
* Roles and sessions security
